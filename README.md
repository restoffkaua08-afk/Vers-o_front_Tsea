# TSEA V-Twin — Front Lab

Protótipo em HTML, CSS e JavaScript para testar uma nova interface antes de migrar para o frontend oficial em React.

## Telas

- Visão Geral
- Operação
- Gêmeo Digital
- Histórico
- Relatórios
- Dados Técnicos

## Como rodar

Use o comando:

py -3 -m http.server 5500

Depois acesse:

http://127.0.0.1:5500

## Observação

Este repositório é apenas um laboratório visual. O sistema oficial continua no repositório tsea-sistema.
