# TSEA V-Twin — Front Lab

Laboratório visual em HTML, CSS e JavaScript baseado na estrutura do frontend oficial do sistema TSEA.

## Telas

- Painel
- Operação
- Gêmeo Digital
- Histórico
- Relatórios
- Parâmetros

## Como rodar

py -3 -m http.server 5500

Depois acesse:

http://127.0.0.1:5500

## Observação

Este repositório é só para testar a nova interface. O sistema oficial continua no repositório tsea-sistema.
